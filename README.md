## Command Line Tool for .Net
```
dotnet new: Creates a new project, configuration file, or other artifacts from a template.
dotnet restore: Restores the dependencies for a given application (this is often run implicitly by other commands like build, run, test, publish, and pack).
dotnet build: Builds a .NET application into an executable assembly.
dotnet run: Runs the application from the source code within the development environment.
dotnet publish: Publishes the application for deployment, creating a directory with the necessary files to run the app in a production environment.
dotnet test: Runs unit tests in the specified project.
dotnet clean: Cleans the build outputs of a project.
```
